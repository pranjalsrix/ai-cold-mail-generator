# 🚀 AI Cold Email Generator

A full-stack AI-powered application that generates personalized **cold emails, LinkedIn DMs, and follow-ups** based on user prompts.

Built using the **MERN stack**, integrated with AI APIs, and fully containerized using Docker.

---

## ✨ Features

* AI-generated cold emails & subject lines
* LinkedIn DM & follow-up generation
* User authentication with OTP verification
* Email history tracking
* Secure API with JWT authentication
* Fully Dockerized setup (Frontend + Backend + DB)
* CI/CD pipeline using GitHub Actions

---

## 🛠 Tech Stack

**Frontend**

* React (Vite)
* Tailwind CSS / shadcn/ui

**Backend**

* Node.js
* Express.js

**Database**

* MongoDB (Atlas)

**AI Integration**

* Groq API

**DevOps**

* Docker & Docker Compose
* GitHub Actions (CI/CD)

---

## ⚡ Quick Start

```bash
# Install all dependencies
npm run install:all

# Start development server
npm run dev
```

* Frontend → http://localhost:5173
* Backend → http://localhost:5000

---

## 🐳 Run with Docker

```bash
docker-compose up --build
```
